Compilation

> make
Pour compiler.

> make clean
Pour effacer les binaires.

> make cclean
Pour effacer les binaires et l'executable.

REQUIERT pour compiler:

Pour Windows: cmake avec les modules FindSDL2...
Un CMakeList.txt adapter...

Pour Unix, les paquets et leurs dépendances suivantes:

libsdl2-dev
libsdl2-image-dev
libsdl2-mixer-dev
libsdl2-ttf-dev

REQUIERT pour executer:

Pour Windows:

Les dlls (binaires) dans le dossier où se trouve snake_io.exe:
libfreetype-6.dll
limpg123-0.dll
libpng16-16.dll
SDL2.dll
SDL2_image.dll
SDL2_mixer.dll
SDL2_ttf.dll
zlib1.dll

Pour Unix:

Les paquets et leurs dépendances suivantes:

libsdl2-2.0.0
libsdl2-image-2.0.0
libsdl2-mixer-2.0.0
libsdl2-ttf-2.0.0
